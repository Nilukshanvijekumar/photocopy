Description

Cost-efficient colour production for all office types

Product Specification

General

  • Copy/Print Process: Indirect electrostatic photographic method
  • Processor: Power PC 440 / Core 667MHz
  • Copying Type: 1 beam laser printing (Memory copy standard)
  • Paper Capacity - Standard (1,200 sheets): With 80 g/m2 / Drawer 1 (550 sheets), Drawer 2 (550 sheets), Stack Feed Bypass (100 sheets)
  • Paper Capacity - Max (3,200 sheets): Drawer 1 (550 sheets), Drawer 2 (550 sheets), Large Capacity Feeder (2,000), Stack Feed Bypass (100 sheets)
  • Paper Size: Drawers: A3/A4/A5-R
  • Paper Size: Stack Feed Bypass: A3/A4/A5-R
  • Paper Size: Large Capacity Feeder (Pedestal type): A4
  • Paper Size: Paper Feed Pedestal: A3/A5-R
  • Paper Weight: Drawers: 64 - 105 g/m2 Stack Feed Bypass (Multi-feed):64 - 209 g/m2
  • Paper Weight: Stack Feed Bypass (Single eed):52 - 209 g/m2
  • Paper Weight: Large Capacity Feeder (Pedestal type): 64 - 105 g/m2, Paper Feed Pedestal: 64 - 105 g/m2
  • Memory size: RAM: 1GB (Unified with Page and Main Memory) HDD: 60GB+ (Shared for all imaging functions)
  • Warm-Up Time: Approx. 20 seconds
  • Duplex: Standard Trayless automatic duplex unit
  • Power Consumption: Max. 1.5kW (115V, 220-240V)
  • Dimensions: W575 x D586 x H756mm, (22.6˝ x 23˝ x 29.8˝)
  • Weight: Approx. 57kg (with drum and developer)

Copier

  • Copy Resolution Printing: 2,400 x 600 dpi with smoothing (Mono) Scanning: 600 dpi x 600 dpi (Colour & Mono)
  • Copy Speed e-STUDIO255: 25 cpm
  • First Copy Out Time: Approx. 4.7 seconds
  • Original Size: A4, A3, B5, A4-R, A5-R, B5-R
  • Original Scan Speed Mono: 45 spm (600 dpi), Colour Doc. Mode: 23 spm (600 dpi) / Mono: 57 spm (300 dpi), Colour Doc. Mode: 43 spm (300 dpi)
  • Toner Type: Cartridge
  • Toner Yield: 24,500 Impressions (100% Recycling Toner System)
  • Power Supply: 115 Volts / 12 Amps
  • Copy Mode: Text, Text/Photo, Photo, Colour Doc, Custom
  • Reproduction Ratio: 25% to 400% in 1% increment (Platen) 25% to 200% in 1% increment (RADF)
  • Multiple Copying: Up to 999 copies

Printing Option

  • Print Resolution: 600 dpi x 600 dpi, 1 bit / 2,400 dpi x 600 dpi, with smoothing
  • Print Speed: e-STUDIO255: 25 ppm
  • Interface: Ethernet (10/100BASE-T), USB 2.0/Hi Speed USB, Wireless LAN (option), Bluetooth (option)
  • Printer Language: PCL6 emulation, PostScript3 emulation, XPS
  • Protocols: TCP/IP (IPV4/V6), IPX/SPX, AppleTalk PAP or EtherTalk, LPR/LPD, IPP w/Authentication, Port 9100, Bluetooth (HCRP/BIP/OPP/FTP), Windows Printing, Netware, AS/400, SAP R/3
  • Drivers: Windows 2000, XP,Server 2003 and 2008, 7, Vista,UNIX, Linux, Mac 05,10.2.4, 10.3, 10.4, 10.5, 10.6.
  • USB Direct Print: PDF, JPEG, Slim PDF, XPS

Scan Option

  • Scan Resolution: 100 dpi, 150 dpi, 200 dpi, 300 dpi, 400 dpi, 600 dpi
  • Scan Speed: 57 spm (in 300 dpi, Mono olour)
  • File Format: JPEG, TIFF, PDF, XPS Slim PDF
  • Authentication: LDAP, SMTP Windows Domain
  • Direct Scan: Scan-to-File, Scan-to-Email, Scan-to-e-Filing, Scan-to-USB

Fax Option

  • Compatibility: Super G3, G3
  • Modem Speed:Max. 33.6kbps
  • Scan Speed: 0.7 seconds/page (A4) or 50 scans / minute (A4)
  • Transmission Speed: Approx. 3 seconds / page
  • Data Compression Method: JBIG, MMR, MR, MH
