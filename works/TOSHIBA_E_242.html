TOSHIBA E-242

Description

Cost-efficient colour production for all office types

Product Specification

General

  • Copy/Print Process: Indirect electrostatic photographic method
  • Copying Resolution: 600 dpi / 256 Gradation
  • Scanning Resolution: 600 dpi
  • Printing Resolution: 2400 dpi x 600 dpi (with smoothing)
  • Copying Speed: 18/18/24ppm
  • Std Copier Memory / Max Copier Memory: 112 MB
  • Max Monthly Copy Volume:77,000 / 77,000 / 90,000
  • First Copy Out Time: 7.6 sec. / 7.5 sec. (eS242)
  • Warm-Up Time: 25 Seconds
  • Multiple Copying: Up to 999 Copies
  • Acceptable Paper Size Trays: Letter to Ledger (17-20 lbs. Bond) Bypass: Statement-R to Ledger (17-90 lbs. Index)
  • Special Media Bypass: Envelope (with Network Print Kit Only)
  • Reduction/Enlargement: 25% to 200%
  • Image Density Control: Text / Text-Photo / Photo
  • Multiple Copying: 1 – 999 copies
  • Reproduction Ratio (Zoom): 25 – 200% in 1% increment
  • Paper Supply: Standard 250-Sheet Cassette / Standard 100-Sheet Bypass / Optional 2nd 250-Sheet Paper Feed Unit / Optional 550-Sheet Paper Feed Pedeestal / Optional 550-Sheet 4th Cassette
  • Document Feeder: Optional 100-Sheet ADF or RADF (ST-R - LD / 17 - 33.8 lbs. Bond)
  • Automatic Duplex Unit: Optional (17-20 lbs. Bond)
  • Dimensions: 23.6” x 25.3” x 18.2” (W x D x H)
  • Weight: Appx. 71lbs.
  • PM Cycle, Drum Yield, Dev. Yield: 77,000 / 77,000 / 90,000 Impressions
  • Toner Type: Cartridge
  • Toner Yield: 24,500 Impressions (100% Recycling Toner System)
  • Power Supply: 115 Volts / 12 Amps

Copier

  • Platen Cover: KA1650PC
  • Automatic Document Feeder: MR2020 (100-Sheet)
  • Reverse Automatic Document: MR3023 (100-Sheet)
  • Automatic Duplex Unit: MD0103 (Stackless)
  • Paper Feed Unit: MY1027 (250-Sheet)
  • Paper Feed Pedestal: KD1022 (550-Sheet)
  • Paper Drawer: MY1028 (550-Sheet)
  • Paper Feed Controller: GH1060 (Required with KD1022 and/or MD0103)
  • External Keyboard: GJ1160 (Required with Fax Kit and/or Network Scan Kit)
  • Harness Kit for Coin Controller: GQ1130

Printing Option

  • Std Printer: GDI Printing
  • Max Print Speed: 18/18/24ppm
  • Interface: USB 2.0
  • Operating Systems: Windows 2000/XP/Vista/7/Server 2003/2008
  • Duplex Printing: Yes (Manual)

Scan Option

  • Standard Scan: Local TWAIN Scan (Toshiba Viewer)
  • Resolution: 300 x 300 dpi / 600 x 600 dpi
  • Formats: BMP, TIFF, Multi-TIFF
  • Max Scan Area: Ledger
  • Interface: USB 2.0
