RICOH MPC-3003

Description

Cost-efficient colour production for all office types

Product Specification

Copier

  • Copying process: Dry electrostatic transfer system
  • Resolution: 600 dpi
  • Multiple copy: Up to 999
  • Zoom: From 25% to 400% in 1% steps
  • Warm-up time: 19 seconds
  • First output speed: Full colour: 7.1 seconds
  • First output speed: B/W: 4.6 seconds

Printer

  • Printer language Standard: PCL5c, PCL6, XPS, PDF direct print, Mediaprint (JPEG/TIFF)
  • Printer language Option: Adobe® PostScript® 3™, IPDS, PictBridge
  • Resolution: 1,200 x 1,200 dpi
  • Interface: Standard: USB 2.0, SD slot, Ethernet 10 base-T/100 base-TX, Ethernet 1000 Base-T
  • Interface Option: Bi-directional IEEE 1284, Wireless LAN (IEEE 802.11a/b/g/n), Bluetooth
  • Network protocol Standard: TCP/IP (IP v4, IP v6)
  • Network protocol Optional: IPX/SPX

Scanner

  • Scan speed: B&W: Maximum 80 ipm
  • Full colour: Maximum 80 ipm
  • Resolution: 100-600 dpi Maximum : 1,200 dpi for TWAIN scanning
  • Bundled drivers:: Network TWAIN
  • Scan to e-mail: SMTP, POP, IMAP4
  • Scan to Folder: SMB, FTP, NCP

Papper Handling

  • Recommended paper size: SRA3, A3, A4, A5, A6, B4, B5, B6
  • Paper input capacity: Maximum: 4,700 sheets
  • Paper output capacity: Maximum: 1,625 sheets

Fax (Option)

  • Circuit: PSTN, PBX
  • Compatibility: ITU-T (CCITT) G3
  • Resolution: Standard/detail: 8 x 3.85 line/mm, 200 x 100 dpi, 8 x 7.7 line/mm, 200 x 200 dpi
  • Resolution: Optional: 8 x 15.4 line/mm, 400 x 400 dpi, 16 x 15.4 line/mm, 400 x 400 dpi
  • Transmission speed: G3: 2 second(s)
  • Modem speed: Maximum: 33.6 Kbps
  • Memory capacity: Standard: 4 MB
  • Maximum: 28 MB

Software

  • Standard: SmartDeviceMonitor™ / Web SmartDeviceMonitor™ / Web Image Monitor
  • Optional: GlobalScan NX / Card Authentication Package / Unicode Font Package for SAP / Remote Communication Gate S Pro

Other Option

  • ADF handle, 1 x 550-sheet paper tray, 2 x 550-sheet paper tray, 2,000-sheet large capacity tray, 1,500-sheet side large capacity tray, Bridge unit
  • Ricoh has designed these products to meet EC ENERGY STAR guidelines for energy efficiency.
